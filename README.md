# Casino Simulator
### C++ Final Project
### Miles Trompeter

### Build Instructions:
* Download "executable.zip"
* Extract All
* Navigate to "C++ Final Project" Directory
* Double click the "CasinoSim.exe" executable

### Description
This is a simple casino simulator with Slots and Blackjack. 
The user can add more chips by adding a credit card number with desired chip amount. (Does not store credit card number)

### Slots
User enters a bet, then clicks spin. The user wins if there are 3 fruits, 3 matching fruits, 3 bells, or 3 7s.

### Blackjack
Standard blackjack gameplay. Normal win pays 1:1, blackjack pays 3:2
